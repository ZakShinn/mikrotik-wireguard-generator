# mikrotik-wireguard-generator
A powerful web-based MikroTik WireGuard configuration generator. Supports QR codes, bulk ZIP downloads, and automated RouterOS scripts. Tiếng Việt: Công cụ tạo cấu hình MikroTik WireGuard mạnh mẽ trên web. Hỗ trợ mã QR, tải ZIP hàng loạt và tự động tạo script RouterOS.
